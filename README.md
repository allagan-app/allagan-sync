# Allagan Sync

Dalamud plugin for [allagan.app](https://allagan.app) — automatically syncs your FFXIV collections and contributes to community-driven statistics.

## What it does

- **Collection Sync** — Detects your unlocked mounts, minions, orchestrion rolls, emotes, titles, achievements, bardings, Triple Triad cards, fashion accessories, facewear, sightseeing log entries, fish, Blue Mage spells, and character customizations, then syncs them with your allagan.app profile.
- **Event Tracking** (opt-in) — Captures in-game activities like desynthesis results and contributes them to community-driven statistics such as drop rates.

## Install as Custom Repo in Dalamud

Use this URL in `Settings -> Experimental -> Custom Plugin Repositories`:

```text
https://allagan.app/dalamud/pluginmaster.json
```

After adding the URL, open `/xlplugins`, search for `Allagan Sync`, and install.

## Setup

1. Create an account on [allagan.app](https://allagan.app) and add your character.
2. Generate an API token on the website.
3. In the plugin, open Settings and paste the token.
4. Your collections will sync automatically.
